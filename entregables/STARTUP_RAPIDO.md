# Startup Rapido

## Opcion 1: Inicio completo

Desde la raiz del proyecto:

```powershell
.\start-complete.ps1
```

## Opcion 2: Inicio por partes

Backend:

```powershell
.\start-local.ps1
```

Frontend:

```powershell
cd rednorte-frontend
npm run dev
```
