# Screenshots de Cobertura y Evidencia

## 1. Pruebas unitarias

Capturar por modulo:

1. Reporte JaCoCo general.
2. Cobertura por clase.
3. Resultado de ejecucion de tests.

## 2. Pruebas de integracion

Capturar:

1. Request y response de casos criticos.
2. Logs de integracion entre servicios.
3. Verificacion de persistencia posterior.

## 3. Pruebas end-to-end

Capturar:

1. Flujo UI de alta de paciente.
2. Flujo UI de alta de cita.
3. Flujo de actualizacion/eliminacion.
4. Evidencia de resultado en API o almacenamiento.

## 4. Hallazgos

Si una prueba detecta defecto, incluir captura de:

1. Estado previo con error.
2. Cambio aplicado.
3. Estado posterior corregido.
