# RedNorte Platform - Gestion de Pacientes y Citas

Sistema de gestion sanitaria con arquitectura de microservicios (Java/Spring Boot) y frontend en React.

---

## 1. Componentes

- rednorte-eureka: descubrimiento de servicios.
- rednorte-bff: gateway y orquestacion.
- rednorte-ms-pacientes: gestion de pacientes.
- rednorte-ms-citas: gestion de citas.
- rednorte-frontend: interfaz web.

---

## 2. Funcionalidad principal

- CRUD de pacientes.
- CRUD de citas.
- Busqueda y filtrado en tiempo real.
- Validaciones y manejo de errores.
